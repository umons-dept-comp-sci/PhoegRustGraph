dnm

Diameter of Hnm with same order and size as the graph given in parameter. Hnm is the graph on n vertices and m edges obtained by taking the biggest clique that does not disconnect the graph and adding a path with the remaining vertices linked to a vertex of the clique. If there are remaining edges, they are added between the closest vertex of the path to the clique and vertices of the clique.

irregularity

Computes the irregularity of a graph. This invariant is defined as the sum for all edges of the absolute value of the difference between the degrees of its extremities. If the graph has no edges, 0 is returned.

is_connected
